leetCode 刷题系列 - 高精度算法

leetCode 刷题系列是为了将自己之前丢失的算法基础 + 算法敏感度弥补回来，在刷 leetCode 的过程中会尽量选择相近类型算法的题目整合在一起，并使用尽可能多的语言来实现它。

在 leetCode 上，第二道题便是中等难度的“高精度计算”问题（加法），先来看一下问题描述：

You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.
Example:

稍微有一点儿英文基础肯定就能明白题目描述的问题：

  1. 题目中说明会给出两个输入“链表”结构的数字描述；
  2. 这两个输入链表都是从低位到高位的顺序；
  3. 要输出一个从低位到高位的符合要求的链表；

解决方案

根据题目描述，这是一个典型的高精度加法考察，其实根据加法的算法过程进行模拟即可；
注意以下几点：

  1. 从低位到高位依次相加；
  2. 判断当前位次是否有进位；
  3. 获得当前位次的和，若有进位则将 +1；
  4. 判断当前进位的和是否大于或等于 10，若满足条件，则取模（%），并将进位标识标志为 true；
  5. 循环加到无下一位位置；
  6. 返回结果链表；
